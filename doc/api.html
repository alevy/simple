API Documentation

The API for each of the libraries is available on Hackage:

  • simple - core web framework
  • postgresql-orm - ORM for PostgreSQL
  • simple-postgresql-orm - connector package for simple and postgresql-orm
  • simple-templates - a simple but powerful templating language.
  • simple-session - cookie-based session management

You can check out the full source code for all the packages:

git clone https://github.com/alevy/simple.git

The repo contains an example app (under the blog-example/ folder).
